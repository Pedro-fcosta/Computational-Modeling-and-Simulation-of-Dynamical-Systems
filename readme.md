# Computational Modeling and Simulation of Dynamical Systems

**Modelagem Computacional e Simulação de Sistemas Dinâmicos**

Undergraduate research repository focused on computational dynamics, numerical simulation, and visualization of dynamical systems using Python.  
Repositório de iniciação científica voltado para dinâmica computacional, simulação numérica e visualização de sistemas dinâmicos utilizando Python.

## English

This repository contains the development of an undergraduate research project focused on the computational modeling and simulation of dynamical systems using Python.

The main idea of this repository is to organize different computational dynamics studies into separate folders, where each folder represents a specific physical system or simulation project.

For example, the repository may include independent folders for:

- simple harmonic oscillator;
- damped oscillator;
- pendulum;
- nonlinear systems;
- other future studies related to dynamics, numerical methods, and chaos.

### Objective

The main goal of this project is to study dynamical systems through computational methods, combining:

- physical modeling;
- programming;
- numerical methods;
- graphical visualization;
- simulation of system behavior over time.

### How to Run

1. Clone this repository:

```bash
git clone https://github.com/Pedro-fcosta/Computational-Modeling-and-Simulation-of-Dynamical-Systems
```

2. Access the project folder:

```bash
cd your-repository
```

3. Enter the folder of the desired project:

```bash
cd simple-harmonic-oscillator
```

4. Install the required libraries:

```bash
pip install numpy matplotlib scipy
```

If the project uses a graphical interface, Tkinter may also be required.
In many Python installations, Tkinter is already included by default.

5. Run the Python file:

```bash
python main.py
```
